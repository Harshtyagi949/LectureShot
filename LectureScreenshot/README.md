# LectureShot — Auto Screenshot App

## App kya karta hai
- Background mein chalta hai
- Har 2 second mein screen check karta hai
- Jab teacher naya content likhta hai (screen change ho) → auto screenshot
- Pictures/LectureShots folder mein save karta hai
- Sensitivity adjust kar sakte ho

## Android Studio mein kaise open karein

1. Android Studio download karo: https://developer.android.com/studio
2. "Open Project" → yeh LectureScreenshot folder select karo
3. Gradle sync hone do (internet chahiye pehli baar)
4. Phone ko USB se connect karo — Developer Options ON karo
5. Run button dabao → APK phone pe install ho jaayega

## Phone pe pehli baar use karna

1. App kholo
2. "Display over other apps" permission do
3. Accessibility Settings mein "LectureShot" enable karo
4. Wapas app pe aao → START dabao
5. Screen capture permission do
6. Ab koi bhi video chalao — screenshot auto-save hote rahenge!

## Sensitivity kya hai

- **5-10%** = har thodi si movement pe screenshot (zyada screenshots)
- **15%** = default — sirf jab teacher naya likhta hai (recommended)
- **30-50%** = sirf bada change hone pe (kam screenshots)

## Screenshots kahan milenge

- Android 10+: App ke private folder mein (Files app → Android → data → com.lecture.screenshot → files → Pictures → LectureShots)
- Android 9 aur neeche: Pictures → LectureShots

## Claude pe bhejne ke liye

1. Gallery mein LectureShots folder kholo
2. Screenshots select karo
3. Claude pe bhejo
4. Likho: "Yeh mere lecture ki screenshots hain, notes banao"
